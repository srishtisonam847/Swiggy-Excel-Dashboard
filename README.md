# Swiggy Sales Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard created using Swiggy sales data.

## Features
- KPI Cards
  - Total Sales
  - Average Rating
  - Average Order Value
  - Rating Count
  - Total Orders

- Interactive Slicers
  - Month
  - Category
  - Restaurant Name

- Visualizations
  - Monthly Sales Trend
  - Daily Sales Trend
  - Weekly Sales Trend
  - Sales by Food Type
  - Top 5 Cities by Sales
  - Sales by State

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- GETPIVOTDATA
